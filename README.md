# SkylinePinz

## Overview

A live geomapping Angular app that allows users and guests to pin messages on a map. Users can append images to their pins, manage their pins, and comment on other users pins.

## Technologies
### FE
* Angular.js
* D3.js

### BE
* Express.js
* PostgreSQL
* MinIO
